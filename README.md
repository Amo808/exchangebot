# 🚀 MCP Server

MCP Server — это API-сервис на **FastAPI**, который предоставляет курсы валют, погоду и финансовые новости.

## 📌 Функции API:
- 📈 **/currency-rate** — Конвертер валют
- ⛅ **/weather** — Погода в городах
- 📰 **/news** — Финансовые новости
